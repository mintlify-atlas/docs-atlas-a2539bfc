  <Steps>
    <Step title="Install the extension">
      Install from the Chrome Web Store or Firefox Add-ons marketplace:
      
      - [Chrome Web Store](https://chromewebstore.google.com/detail/jmlcckldioeofjchfaomgaimhpnlfgok)
      - [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/freshjuice-hubspot-devtools/)
      
      The extension will appear in your browser toolbar with the FreshJuice icon.
    </Step>
    
    <Step title="Navigate to a HubSpot page">
      Open any HubSpot CMS page you're working on. The extension automatically detects HubSpot domains.
    </Step>
    
    <Step title="Toggle debug parameters">
      Click the extension icon and toggle on any debug mode:
      
      - **Debug Mode** - Adds `hsDebug=true` to see HubL variable output
      - **Cache Buster** - Adds `hsCacheBuster={timestamp}` to bypass caching
      - **Developer Mode** - Adds `developerMode=true` for backend access
      
      The page will reload automatically with the selected parameters applied.
    </Step>
    
    <Step title="Explore features">
      The extension includes:
      
      - Auto-apply debug parameters to internal links
      - Right-click context menus for quick access
      - Keyboard shortcuts for power users
      - Badge indicator showing active parameters
      
      <Note>
      All settings are synced across your browser instances and persist across sessions.
      </Note>
    </Step>
  </Steps>
